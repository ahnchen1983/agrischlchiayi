# 植劇場：一個做筆電的人，如何種下台劇復興的種子

> **30 秒概覽：** 2016年8月，八部台灣電視劇同步開播，主創陣容包含蔡明亮、陳玉勳等殿堂級導演，演員卻清一色是剛受訓的菜鳥。出資人是和碩科技董事長童子賢，一個以製造筆電代工起家的科技人。植劇場的邏輯反直覺：把最頂的師傅配給最嫩的徒弟，把資源砸在「種子」而非「果實」上。兩年後，24位新演員裡走出了[[許光漢]]、陳妤等名字；台劇產業也多了一套證明：多元類型劇在台灣不是不能活。

---

2014年底，導演王小棣打電話給七位老朋友，問的問題很直接：「你願不願意花時間帶新人？」

電話另一端分別是蔡明亮、陳玉勳、瞿友寧、許傑輝、徐輔軍、王明台、安哲毅，全是在[[台灣電影]]與電視圈累積數十年資歷的導演群。在那個時代，找這幾個人拍戲，沒有人會說「帶新人」，合理的用法是「拍重磅作品」。

但王小棣說的就是帶新人。

她從300多位報名者中，選出24位被稱為「好植之徒」的潛力新人，先送進她主導創立的「Q Place 表演教室」，從京劇訓練到小丑表演，讓現代人的身體重新被傳統劇場的紀律校準。這24人受訓之後的第一個實際戰場，就是植劇場（Q Series）的八部戲。

## 一個做筆電的人的文化賭注

整套計畫需要資金。那筆資金，來自一個乍看毫不相干的人：和碩聯合科技董事長童子賢。

在台灣科技業，童子賢以共同創辦華碩、後來帶領和碩做 Apple 代工起家，是典型的硬體製造業大老。他同時支持過誠品書店多年、贊助[[雲門舞集]]、投資《一把青》、捐獻楊牧文學獎，在科技業與文化藝術之間來回跨界是他的個人風格。

2016年，他以旗下藝碩文創出資，讓植劇場成真。

這個決定從商業邏輯來看匪夷所思。找最資深的導演搭配最沒名氣的演員，在當時韓劇陸劇夾擊的市場環境下，既不保證收視，也不保證回收。童子賢事後被問及動機，公司主管轉述他的態度是「對藝文、創意與人才培養的熱愛從沒停過」，這個計畫對他而言更像是一種對台灣文化生態的長期投資，而非短期獲利的計算。

王小棣對這個邏輯的解釋則更直接。她在 BIOS monthly 的訪談中說：「這遊戲已經不好玩了！所以我們希望這個遊戲更有意思。」

> 📝 **策展人筆記**
> 把最貴的師傅配給最嫩的徒弟，反而是一種人才培育的加速器。學徒在最短時間內被拉進最高標準的工作環境，這不是慈善，是一種高效率的知識轉移。台灣影視業缺乏的從來不是才華，而是這樣的工作機會。

## 八部戲，四種類型

植劇場的完整格局是 8 部劇、52 集，分為四個類型，每類型兩部，2016年8月至2017年8月輪番在台視、八大電視、公視播出。

| 類型     | 作品                   | 主要演員                     | 備注                                        |
| -------- | ---------------------- | ---------------------------- | ------------------------------------------- |
| 愛情成長 | 戀愛沙塵暴             | 吳慷仁、柯淑勤、許光漢、陳妤 | 第52屆金鐘9項入圍，柯淑勤奪后、陳妤獲新人獎 |
| 愛情成長 | 荼蘼                   | 楊丞琳                       | 徐譽庭編劇，首爾國際電視節、亞洲電視大獎    |
| 愛情成長 | 姜老師，妳談過戀愛嗎？ | 藍正龍、許光漢               | 王小棣、柯雁心聯合編劇                      |
| 驚悚推理 | 天黑請閉眼             | 楊謹華、張書豪、徐鈞浩       | 柯貞年、陳玉勳聯合導演                      |
| 驚悚推理 | 積木之家               | 鄭人碩、禾浩辰               | 懸疑心理類型                                |
| 靈異恐怖 | 夢裡的一千道牆         | 柯震東、謝欣穎               | 恐怖驚悚路線                                |
| 原著改編 | 花甲男孩轉大人         | 盧廣仲、嚴正嵐               | 改編楊富閔小說，大受歡迎後推出電影版        |
| 原著改編 | 五味八珍的歲月         | 歸亞蕾、傅孟柏               | 改編廚藝大師傅培梅自傳                      |

這個四類型的設計，本身就是一種宣言：台灣電視劇不應該只有偶像劇。在當時，「台劇」幾乎是「偶像劇」的同義詞，植劇場用八部風格各異的劇集，強制拓寬了這個定義。

## 戀愛沙塵暴裡的新臉孔

2016年，一個剛從歌唱訓練生轉跑道的年輕人，拿到植劇場的角色通知。他在《戀愛沙塵暴》裡飾演花心學長莊浩洋，同年又在《姜老師，妳談過戀愛嗎？》中挑戰性衝動遲緩兒威政，一個需要高難度肢體演技的角色，拍攝期間讓他坦言「非常煎熬」。

那個年輕人叫許光漢。《姜老師》播出後，他入圍第52屆金鐘獎男配角獎，後來憑《想見你》成為台劇新一代國民男友，2024年又接到好萊塢《尚氣2》的邀約。整條軌跡的起點，是植劇場給的那兩個角色。

植劇場的出道故事不只屬於他一個人。據 Elle Taiwan 2024年報導，許光漢在植劇場之前「有一餐沒一餐的日子，讓家人都相當擔心，甚至勸他直接找一份工作，不過他卻決定在這條不確定的路上等待機會，之所以能夠堅持下來，完全是靠著一股衝動和天真。」

同樣出發自植劇場的陳妤，憑《戀愛沙塵暴》一舉拿下金鐘新人獎，往後在多部台劇中確立了自己的位置。植劇場的邏輯在這裡得到了驗證：你不需要用已有名氣的人，你需要給素人一個正確的工作環境。

## 花甲男孩讓台中農村成為舞台

八部戲裡，大眾接受度最廣的是《花甲男孩轉大人》。改編自作家楊富閔的同名小說，由瞿友寧執導，歌手盧廣仲跨界主演28歲魯蛇青年鄭花甲。

在韓劇和陸劇當道的2017年，這部以台中農村為背景、用台語與國語混搭、講阿嬤彌留家族亂象的本土喜劇，開播首集收視率衝到1.22%。在當時台劇普遍不振的環境下，這個數字讓業界重新相信台灣觀眾還是買台灣故事的單。劇集的成功促成了2018年的電影版《花甲大人轉男孩》，原班人馬春節上映，把當初的電視實驗轉化成院線商業成品。

> 📝 **策展觀點**
> 《花甲男孩轉大人》的每一個決策都不是安全牌：選了一個沒演過戲的歌手、拍了一個台中農村的家族故事、在韓劇陸劇橫掃的市場裡硬出。正因如此，它成功的意義遠大於它的收視率數字，它是一個示範：本土題材、素人主角、非偶像劇類型，三樣加在一起不等於「不能播」。

## 金鐘成績單

兩屆金鐘獎，植劇場累積了這樣的成績：

| 屆次             | 入圍     | 得獎    |
| ---------------- | -------- | ------- |
| 第52屆（2017年） | 24項     | 5項     |
| 第53屆（2018年） | 15項     | 4項     |
| **合計**         | **39項** | **9項** |

值得注意的是，這些成績由不同作品分攤，沒有一部戲包攬所有獎項。這驗證了植劇場「多元類型」的初衷：不是靠一部神劇撐場面，而是用整個系列展示台劇的寬度。第53屆金鐘獎，《花甲男孩轉大人》奪得戲劇節目獎、男主角獎等大獎，盧廣仲的演技跨越了「歌手客串」的框架，被評審正式肯定為戲劇表演。

## 遊戲規則確實不好玩了

植劇場結束後，台灣影視業的討論並未停歇。批評者指出，這個計畫的商業模式難以持續：以頂尖導演帶菜鳥演員的成本結構，在沒有外部大金主的情況下難以複製。植劇場後期傳出編劇薪資爭議，王小棣在對外聲明中表示帳目有完整紀錄，顯示即便是理念先行的計畫，仍難免產業結構性矛盾的衝擊。

對於「她試圖改變遊戲規則」的說法，王小棣本人並不買單。她說她並不刻意要當 Game Changer，「這遊戲已經不好玩了！所以我們希望這個遊戲更有意思。」（Elle Taiwan，2022年）

這是一個重要的區別。植劇場的動機不是從上而下的「產業改革宣言」，而是對既有框架的厭倦，以及「試試看別的」的衝動。這種心態下做出來的東西，往往比刻意的革命更有穿透力。

## 茁劇場：種子的第二季

2021年，植劇場以「茁劇場」之名回歸。王小棣再次坐鎮總監製，但這次的格局與第一季截然不同。茁劇場的導演群加入了更多中生代面孔，曾經是「好植之徒」的演員開始反過來成為新一批學員的學長姐。這個循環本身就是植劇場最初設計的長期企圖：不是做一季就結束的電視節目，而是建立一條可以持續運轉的人才輸送帶。

被植劇場種下的那批種子，各自長成了不同的樣貌。[[許光漢]]從植劇場的配角出發，經過《想見你》的全民關注，到2024年接到好萊塢《尚氣2》邀約，走出了一條素人演員幾乎不可能走出的國際路線。陳妤從金鐘新人獎開始，在《你的孩子不是你的孩子》、《我們與惡的距離》等作品中持續發展，成為台劇品質保證的常見面孔。嚴正嵐則在《花甲男孩轉大人》之後，轉進電影與舞台劇雙軌發展。

> 「有一餐沒一餐的日子，讓家人都相當擔心，甚至勸他直接找一份工作，不過他卻決定在這條不確定的路上等待機會，之所以能夠堅持下來，完全是靠著一股衝動和天真。」——Elle Taiwan 報導許光漢植劇場時期的生活（2024年）

這些個人軌跡的背後，有一個更大的結構變化：在 Q Place 表演教室存在之前，台灣缺乏系統性把素人磨成專業演員的機制。演員養成要不是靠科班訓練（如台藝大、北藝大戲劇系），就是靠偶像劇選角的運氣。植劇場和 Q Place 提供了第三條路線，一條結合密集實戰與業界頂級師資的路線，讓沒有學院背景的人也能被看到。這個架構本身，比任何一部劇的收視率更有長期價值。

## 不是革命，是厭倦之後的行動

回頭看，2016年前後是台灣戲劇轉型的關鍵年份。同一時期，Netflix 開始投資台灣原創內容，《[[通靈少女]]》、《麻醉風暴》相繼出現，台劇開始被國際串流平台注意到。但植劇場不屬於這條串流資本的脈絡。它靠的是一個科技業老闆的文化信念、一群資深導演的不甘心，以及一座名叫 Q Place 的小小表演教室。八部戲在台視和八大的深夜時段悄悄播出，收視率從未驚天動地，卻在金鐘獎上累積了39項入圍、9座獎盃的成績單。

更關鍵的是，它在台灣影視業留下了一個問題：如果頂尖導演願意花時間帶新人，如果科技業的資金願意進入文化領域，如果電視台願意讓出黃金時段給非偶像劇的類型實驗，台灣的戲劇生態會長成什麼樣子？

植劇場沒有完整回答這個問題。它的商業模式確實難以複製，它的編劇薪資爭議也暴露了理念與現實之間的裂縫。但它種下的東西已經超出了一個電視節目的框架：有些長成了演員，有些長成了觀眾對台劇的信心，有些則長成了後來者可以依循的路徑。

原來台劇可以不只是這樣。
