# 農產品加工策略

> **一句話核心洞見**：加工不是「多賺一道」，而是農場結構轉型——它把鮮品的「不可控因子」（規格、損耗、淡旺季、議價弱勢）轉成「可控資產」（標準化、保存期、全年現金流、品牌敘事）；但這條路要橫跨「研發 / 法規 / 包裝 / 通路」四個全新領域，多數成功案例倚賴創辦人個人多才多藝，因此可以複製方法論，難以複製人。

## 1. 為什麼要做加工

樣本中 20 位有加工實作或規劃的農友，加工背後的功能可歸納為四種：

### 1.1 等級分配（最常見、效益最直接）

把同一批鮮品依品質分級：

```
A 級貨 ──▶ 直銷宅配 / 禮盒（高單價、品牌承載）
B 級貨 ──▶ 行口 / 批發（去化主流量）
C 級貨 ──▶ 加工原料（NG 果、規格外、保存壓力大者）
```

效果：**避免 C 級貨低價賤賣或丟棄、提高每批產量的整體 ARPU**。一位洋香瓜+番茄+洋桔梗多作物農友（年收約 290 萬）就用這個三層策略管理每批採收。

### 1.2 季節性平滑

鮮品集中採收 → 加工品全年出貨。最典型是烏魚子（年節 2 個月集中銷售）、果乾、果醋、米製品。透過加工，把「3 個月旺季 + 9 個月空檔」改成「全年現金流」。

### 1.3 通路升級

鮮品很難走超商、伴手禮店、企業禮品；加工品可以。這是唯一把農場帶進「精品禮品市場」「企業 B2B 訂單」的路徑。

### 1.4 故事承載

鮮品的故事傳達能力很有限（消費者買的是品項、不是農友）。加工品則是天然的故事載體 — 包裝、品牌、來源、手作程序都成為敘事元素。

## 2. 嘉義樣本的加工類型分布

20 個有加工或六級化規劃的案例：

| 類型 | 案例數 | 代表品項 |
|---|---|---|
| 六級化整合（多項加工 + 體驗 + 電商） | 15 | 跨類別 |
| 茶葉 / 咖啡精製 | 5 | 金萱、烏龍、咖啡熟豆 |
| 烏魚子 / 水產加工 | 3-4 | 烏魚子、一夜干、魚鬆、滴魚精 |
| 鮮果加工 | 3 | 果乾、火龍果鳳梨酥、果醋、果酒 |
| 米 / 雜糧加工 | 2 | 米香、Q寶粉、麥芽糖、本味醂 |
| 調味料 / 醬料 | 2-3 | 山葵醬、白玉酥、辣椒醬、煲湯 |
| 蛋品加工 | 1 | 鵪鶉蛋（HACCP 一條龍） |
| 竹筍初級加工 | 1 | 筍乾、依部位分類銷售 |

> 註：總和大於 20 是因為許多案例同時做多種加工。

## 3. 五大主流加工路徑

### 3.1 路徑 A：鮮果加工（果乾 / 果醋 / 果酒 / 烘焙鳳梨酥）

**典型配置**：

- 主作物：芭樂 / 芒果 / 龍眼 / 蓮霧 / 火龍果 / 鳳梨
- 加工品線：原汁、果乾、凍乾、吸凍飲、鮮果蛋捲、果醋、軟糖、雪酪
- 通路：電商（東森得易購、台灣好農）+ 北部市集（希望廣場、花博、勤美草悟道）+ FB 社團

**典型案例邏輯**：

> 一對情侶共同創辦的水果農場（芭樂 0.4 公頃、芒果 0.2 公頃、龍眼 0.1 公頃、蓮霧 0.075 公頃，年收約 80 萬，100% 直銷）— 把「所有流行的農業轉型元素」（IoT 感測、二級加工、食農教育、多平台電商）都試了一遍。

**關鍵啟示**：理論完整度 ≠ 商業執行力。SKU 太多會分散有限的時間和廣告預算。

### 3.2 路徑 B：水產加工（一夜干 / 魚鬆 / 滴魚精 / 烏魚子）

**典型配置**：

- 養殖：虱目魚 / 白蝦 / 烏魚 混養
- 加工：一夜干、虱目魚柳、魚鬆、滴魚精；烏魚則加工成烏魚子（生子 + 熟子）
- 通路：粉絲專頁 + 蝦皮 + LINE: redacted + 嘉義優鮮政府品牌 + 旅行社/飯店 B2B

**現金流結構**：

- 烏魚子：強季節性（農曆年前 2 個月為主）、單價 1,000-2,500 元/兩級距、年節企業禮品為高利潤客戶
- 一夜干 / 魚鬆：全年出貨，通路相對穩定

詳細水產加工脈絡見 [[嘉義水產養殖經營實務]]。

### 3.3 路徑 C：米 / 雜糧 / 加工矩陣（自然農法理念型）

**典型配置**：

- 黑秈米 / 糯米 / 黑豆 / 綠豆 / 芝麻
- 加工：米香、Q 寶粉、本味醂、古法柴燒麥芽糖、芝麻清油、麻糬、紅龜粿
- 通路：宅配 70% + 主婦聯盟合作社 + 食光機課程 + 鄰菜等理念商家

**關鍵啟示**：

- **食品工廠資格限制是天花板**：手作量級無法正式上架超商或大型通路
- **理念農業的客群天然黏著但難擴散**：消費者忠誠度高，但要「跨出主婦聯盟生態系」很困難

### 3.4 路徑 D：精品調味料 / 醬料

**典型配置**：

- 主作物：苦瓜 / 火龍果 / 山葵 / 香檬 / 椒類
- 加工：白玉酥、白玉茶、火龍果鳳梨酥、山葵醬、香檬辣椒醬、港式煲湯、蜜香檬飲
- 認證：HACCP / 有機 / SGS（依品項）

**關鍵啟示**：

- 調味料 SKU 多會稀釋資源 — 一個案例同時做 5+ 種加工品，但年收僅 21 萬
- 反之單品聚焦的精品定位（米其林餐廳指名山葵）可達 ~290 萬年收

### 3.5 路徑 E：規模化蛋品 / 工業級加工

**典型配置**（樣本中極少數）：

- 鵪鶉蛋三代家業：自有 + 契作牧場 6 大牧場、1.8 公頃、年產 1,800 萬+ 顆
- 台灣唯一合法 HACCP 鵪鶉蛋加工廠
- 日本全自動設備（自動餵食、撿蛋、煮蛋、剝殼）
- 產品線：生蛋、水煮蛋、滷蛋（多口味）、皮鹹蛋；研發中：冰淇淋、蛋捲、布丁
- 數位化：正航 ERP

**關鍵啟示**：

當一個地方產業（嘉義溪口鵪鶉蛋）有 HACCP 合法加工廠 + 日本級設備 + ERP 系統時，創辦人的角色已從農友轉變為**產業代言人**（推動「鵪鶉蛋元年」、訂定「台灣鵪鶉日」等）。這是極少數小農做得到的路徑。

## 4. 規模對應加工模式

不同年收級距，加工的可行模式截然不同：

| 年收區間 | 可行加工模式 | 典型品項 | 風險點 |
|---|---|---|---|
| 微型（< 50 萬） | 自製手工 + 市集 + 電商試水 | 果乾、米香、辣椒醬 | SKU 過多 / 資金緊 / 法規灰色 |
| 小型（50-150 萬） | 1-2 個招牌品項 + 在地代工 | 一夜干、果醋、單品禮盒 | 產線品控 / 通路依賴 B2B |
| 中型（150-400 萬） | 委託代工 + 自有品牌 | 烏魚子、火龍果鳳梨酥、白玉酥 | 品牌冷啟動 / 季節性現金流 |
| 中大型（400-700 萬） | 部分自加工 + B2B 禮品 | 茶葉精製、苦瓜六級化 | 整合複雜度 / 雇員管理 |
| 大型（> 1000 萬） | 自建 HACCP 廠 + ERP + 自動化 | 鵪鶉蛋、火龍果禮品、水產一條龍 | 資本回收期 / 法規 |

**關鍵原則**：**不要跳級**。從 A 級分配 → 1 個招牌品項 → 多品項 → 自建廠房 → 工業化，每級需要 1-3 年驗證。

## 5. 三個關鍵決策

### 5.1 自建 vs 代工 vs 品牌貼牌

| 模式 | 投入 | 控制力 | 適用情境 |
|---|---|---|---|
| 自建小型加工坊 | 50-200 萬 + 證照 | 高（配方、產期、包裝皆自主） | 量小、SKU 少、品質敏感（如山葵醬） |
| 委託代工 | 月幾萬到幾十萬 | 中（配方自有、產線他人） | 量已經穩定、想專心通路（如果乾、米香） |
| 品牌貼牌 | 採購成本 + 品牌投入 | 低（產品非自製） | 想做純品牌路線（如「自然本膳」型） |

樣本中**最常見的失誤**：太早自建廠房。建廠等於把現金流壓在資本回收期上，對中小規模農友是嚴重風險。

### 5.2 一個招牌品 vs 多品項組合

| 策略 | 優點 | 缺點 |
|---|---|---|
| 一個招牌品項 | 資源聚焦、深耕容易、品質穩定 | 季節性風險集中 |
| 多品項組合（5+） | 全年出貨、組合禮盒議價力 | SKU 管理成本高、品控分散、行銷預算稀釋 |

**建議**：**先 1 個招牌品做透，再加第 2 個**。樣本中年收 21 萬的「自然本膳」案例做了 5+ 種加工品，反而年收最低 — 過度多元化是陷阱。

### 5.3 B2C 直銷 vs B2B 企業禮品

| 通路 | 客單價 | 訂單穩定度 | 行銷投入 |
|---|---|---|---|
| B2C 直銷（粉絲頁、電商、市集） | 低-中 | 不穩 | 高（內容、廣告、社群維護） |
| B2B 企業禮品（公司送禮、CSR、KA） | 高 | 高（年訂或季訂） | 中（人脈、提案、樣品） |

樣本案例：

- 有機火龍果案例做出明基友達「指定精品禮品」訂單 → B2B 進階
- 烏魚子案例若能開發 B2B 企業送禮，可解決季節性現金流不均

**建議**：產品穩定後，**主動開發 B2B 通路**比繼續打 B2C 廣告 ROI 更高。

## 6. 法規與認證

加工的法規門檻是新進農友最容易低估的成本：

### 6.1 必要法規（涉及食品安全）

- **食品業者登錄**：年銷售額 > 8 萬即須登錄
- **食品工廠登記**：自建加工廠必備
- **HACCP**：水產、肉品、蛋品的工廠級加工
- **食品 GMP**：自願性，但部分通路要求

### 6.2 加值認證（提高溢價）

- **產銷履歷**：成本約 1-3 萬/年，多數通路認可
- **有機認證**：成本 5-15 萬/3 年，溢價 30-100%
- **CAS / TAP**：政府背書、出口可用
- **嘉義優鮮**：縣府品牌，免費 + 集體行銷

### 6.3 關鍵盲點

- **網路販售也適用食安法**：粉絲頁/蝦皮上的加工品，仍須登錄
- **代工也須代工廠合格**：用合法代工廠才能避責任轉嫁

## 7. 擴散障礙（為什麼成功案例難複製）

整理 20 個案例的「嘗試過的解法」與「為什麼沒有擴散」，共通障礙有六：

1. **食品工廠資格門檻**：手作合法量極小，正式量產需建廠
2. **加工技術經驗化**：配方依賴個人手感，難以標準化授權
3. **包裝設計專業**：印刷、設計、成本對小農是新領域
4. **多通路管理時間**：粉絲頁、蝦皮、官網、市集全部要顧
5. **品牌冷啟動資金**：品牌建立需 2-5 年持續投入
6. **季節性 SKU 衝突**：水果有季節性、加工品有保存期，搭配不易

## 8. 給想做加工的農友的決策樹

```
要做加工嗎？
│
├─ NO（先把鮮品做透）
│    ↓
│   做 A/B/C 等級分配（不需要加工就能提升 ARPU）
│
└─ YES
     │
     ├─ 我目前年收 < 100 萬？
     │    ↓ YES
     │   只做 1 個招牌品（自製或在地代工，市集 + 電商試水）
     │
     ├─ 我目前年收 100-300 萬？
     │    ↓
     │   委託代工 + 自有品牌（2-3 個品項 + B2B 試水）
     │
     ├─ 我目前年收 300-700 萬？
     │    ↓
     │   評估自建小型加工坊（單品深耕 + B2B 進階）
     │
     └─ 我目前年收 > 700 萬？
          ↓
         考慮自建 HACCP 廠 + ERP + 自動化（產業代言人路徑）
```

**任何階段共通的檢查項**：

- [ ] 法規盤點清楚（食品登錄 / 工廠登記 / 必要認證）
- [ ] 包裝設計與成本計算到位（不是選最美，是算 ROI）
- [ ] 通路試水：先 1-2 個通路驗證，再擴張
- [ ] 預留品牌冷啟動 2-5 年的資金緩衝

## 9. 數位化與 AI 機會

從樣本看，加工型農友最需要的工具是：

- **多平台訂單整合**（蝦皮 + 官網 + LINE: redacted + 政府品牌平台）
- **庫存與保存期管理**（加工品保存期短，需要先進先出）
- **B2B 客戶 CRM**（年節提醒、舊客戶回購）
- **包裝設計輔助**（AI 生成包裝視覺、文案）
- **食品法規查詢**（不同品項的法規不同，混亂）

## 11. 結語

加工不是「多賺一道」，而是農場結構轉型——把鮮品的不可控因子轉成可控資產。但這條路橫跨研發 / 法規 / 包裝 / 通路四個全新領域，**多數成功案例倚賴創辦人個人多才多藝**。

對新進農友的建議：**先把 A/B/C 等級分配做好，再選 1 個招牌品深耕，最後才考慮自建廠或多品項擴張**。不要跳級、不要一次全做、不要低估法規與品牌冷啟動成本。

對地方產業組織者：嘉義缺**集體加工廠**和**集體品牌平台**——讓中小農共用 HACCP 廠房、共用 ERP、共用通路議價，這比個別農友各自建廠更有結構效益。
