# Taiwan.md 的起源故事

> **30 秒概覽：** Taiwan.md 是一個由台灣新媒體藝術家**吳哲宇**在 2026 年 3 月發起的開源台灣知識庫，涵蓋歷史、文化、美食、科技、自然等十二個面向，中英雙語、CC BY-SA 4.0 授權完全開源。發起的動機是 2024 年威尼斯雙年展一位義大利策展人的一句提問：「Where can I learn about Taiwan? Like, really learn?」這個網站，是那個提問兩年後的答案。

## 威尼斯的那個提問

2024 年春天，第 60 屆威尼斯雙年展 Personal Structures 單元。

台灣新媒體藝術家吳哲宇帶著他的作品《靈魂魚》參展。開幕酒會上，一位義大利策展人走過來，問了一句很簡單的問題：

> **「Where can I learn about Taiwan? Like, really learn?」**

他當下沒有答案。

台灣的資料當然不是沒有——維基百科有條目、Lonely Planet 有旅遊指南、學術論文有研究。但對一個真心想「認識」這座島嶼的外國人來說，沒有一個以台灣為主體、有溫度、深度夠、持續更新的入口。

更關鍵的是——**當 AI 從網路學習「台灣是什麼」的時候，它學到的是誰寫的台灣？**

如果高品質的英文內容大量由非台灣視角產出，AI 學會的「台灣」就不是住在這裡的人會認出來的台灣。

這不是身份政治問題，是**知識主權問題**。

那句提問在他身體裡埋了兩年。

## 為什麼是 Markdown

2026 年 3 月，他發起了 Taiwan.md。

名字本身就是答案：用最簡單、最開放、最容易被 AI 訓練模型讀取的格式，來寫台灣。

- **.md** 是 Markdown 的副檔名，純文字格式，任何 AI、爬蟲、編輯器都能直接讀
- **CC BY-SA 4.0 授權**，完全開源，可被複製、翻譯、再利用
- **GitHub 協作**，任何人都可以提 PR 補充、糾正、擴充
- **中英雙語優先**（後續擴展到日、韓、法）

這不是一個觀光網站，不是百科全書，是**台灣的數位人類學博物館**。

## 這不只是一個人的專案

Taiwan.md 啟動後，累計超過 900 個 GitHub star、超過一百個國家的訪客、數十位貢獻者提交 PR。臺史博館長**張隆志**公開背書，**唐鳳**為之引薦國際連結。Portaly 贊助平台在上線第一個月開通。

每一篇文章底下的 contributors 欄位都記錄了共同寫作的人。這座「博物館」的真正樣子，是一個**持續呼吸的開源生態**——不是一個人的作品集，是一群願意讓台灣被好好說一次的人共同的 Markdown 檔案。
