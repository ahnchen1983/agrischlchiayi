# 小黃瓜設施栽培技術指南

## 導言

小黃瓜是嘉義地區設施蔬菜的重要經濟作物，因其生長週期短（12-14 週）、年產量高（75+ 噸/0.25 公頃）、銷售通路多元（批發、行口、直銷），已成為許多農友的主要收入來源。本指南以實地農友經驗為基礎，提供從設施規劃、種苗管理、肥水控制到病蟲害防治的完整技術路徑。

---

## 第 1 章：品種選擇與育苗準備

### 適合嘉義地區的小黃瓜品種

嘉義地區的設施小黃瓜栽培主要集中在東石、朴子一帶，常見品種及其特性如下：

#### 日本進口品種（推薦，市場主流）
| 品種 | 外觀特色 | 生育期 | 適栽季 | 產量特性 | 建議規模 |
|------|---------|--------|--------|---------|---------|
| 夏鄉 | 短棘、深綠 | 50-60 天 | 全年 | 穩定高產（280 kg/100m²） | 推薦首選 |
| 新輝 | 棘少、中長 | 60-70 天 | 春秋季 | 耐熱性強 | 夏季搭檔 |
| 陸奧 | 長條、淺綠 | 65-75 天 | 秋冬季 | 低溫生育佳 | 冬季專用 |

**選購建議**：
- 向種苗廠商索取近 3 年當地栽培數據
- 優先選擇當地農友「驗證過」的品種（降低失敗風險）
- 購買通過檢疫的種苗，避免病害潛伏

### 育苗流程（5-7 週）

#### 育苗期重點時程
```
第 1-2 週：發芽期
  ✓ 種子浸泡 6-12 小時（28°C溫水）
  ✓ 播種介質：發芽床使用泥炭或椰纖（保濕性優）
  ✓ 溫度：25-30°C，濕度 70-80%
  ✓ 出苗期：播種後 4-7 天

第 3-5 週：生長期
  ✓ 展開真葉 2-3 片
  ✓ 移至本育苗格（50 格或 72 格扦盤）
  ✓ 光照：14 小時日照（不足時補充 LED）
  ✓ 溫度：白天 25°C，夜間 18-20°C
  ✓ 施肥：高磷肥（P:K = 1:2），促進根系發育

第 6-7 週：硬化期
  ✓ 葉片 4-5 片，開始現蔓
  ✓ 降低澆水頻率，讓植株適應乾旱
  ✓ 降低夜溫至 15-18°C，增進光合作用
  ✓ 定植前 1 週停止施肥，促進根系吸收養分
```

#### 常見育苗問題排除
| 問題 | 症狀 | 原因 | 解決方案 |
|------|------|------|---------|
| 徒長 | 苗高 > 15 cm，莖細 | 光照不足 | 增加 LED 補光至 14 小時 |
| 根腐病 | 根部軟爛，枯萎 | 濕度過高，排水不良 | 改用泥炭+珍珠石（1:1），控濕度 60% |
| 病毒病 | 葉片皺縮，失綠 | 環境溫度波動 | 控制溫度 ±2°C 內 |

## 第 2 章：設施規劃與環控系統

### 設施規格（以 0.25 公頃為標準小農規模）

小黃瓜對設施要求不如番茄嚴苛，但環控系統的投資能大幅提升產量穩定性。以下為實地農友的標準配置：

#### 溫室結構
```
土地面積：0.25 公頃（2,500 m²）
實際種植面積：約 1,800 m²（含走道、設備空間）

建築規格：
  - 寬度：8-9 公尺（利於機械操作）
  - 長度：約 200+ 公尺（視地形而定）
  - 高度：2.8-3.2 公尺（便於行走、採收）
  - 跨度：5-6 排（便於人工操作）

骨架材料（成本較低方案）：
  - 簡易鋼骨架 + 農用塑膠布
  - 預計壽命：5-7 年
  - 年折舊成本：約 2-3 萬元
```

#### 環控設備清單（實際案例參考）
| 設備 | 功能 | 單位成本 | 0.25ha 總成本 | 年化成本 |
|------|------|---------|---------------|---------|
| 覆蓋幕 | 遮陰、保溫 | 20 元/m² | 3.6 萬 | 0.5 萬 |
| 側窗開閉器 | 通風、降溫 | 800 元/個 | 4 萬 | 0.5 萬 |
| 灌溉主管組 | 水肥灌溉 | - | 2.5 萬 | 0.3 萬 |
| 溫濕度計（自動）| 監測環境 | 3,000 元/個 | 0.9 萬 | 0.1 萬 |
| 淋浴頭/滴灌帶 | 精準灌溉 | 15 元/個 | 1.8 萬 | 0.2 萬 |

**環控系統總成本估算**：約 12-15 萬元  
**年化環控成本**：約 1.6-2 萬元

### 環控策略（月份調整）

小黃瓜對溫、濕度敏感，不同季節的環控重點不同：

#### 春季（3-5 月）：升溫期
```
目標溫度：白天 25-28°C，夜間 18-20°C
濕度目標：65-75%
通風策略：
  - 上午 8 點開側窗，保持空氣流動
  - 下午 2-4 點開頂窗，排除地表濕熱
  - 下午 5 點後關閉所有窗口

病害防控：
  ⚠️ 春季是白粉病、疫病的高發季
  - 保持 65% 濕度上限（防黴菌）
  - 清晨噴施預防性殺菌劑（每 7 天一次）
```

#### 夏季（6-8 月）：降溫期
```
目標溫度：白天 25-30°C，夜間 20-22°C
濕度目標：60-70%
遮陰策略：
  - 使用 50% 遮光幕（6-8 月）
  - 上午 9 點展開，下午 4 點收起
  - 避免過度遮光（<40%）導致產量下降

通風極為關鍵：
  - 全日開側窗，保持空氣對流
  - 夜間保持 1-2 個通風口，防露水結露
  - 下雨時密閉通風口，防止積水

病害防控：
  ⚠️ 夏季是炭疽病、細菌性病害的高發季
  - 定期檢查葉片下表面
  - 移除病葉，避免二次感染
```

#### 秋季（9-11 月）：穩定期
```
目標溫度：白天 25-26°C，夜間 18-19°C
濕度目標：65-75%
遮陰策略：
  - 9 月中旬開始收起遮光幕
  - 10 月全面停用遮光幕
  - 依晴雨天氣靈活調整

病害防控：
  - 秋季是病蟲害的相對低發季
  - 維持定期防治（10-14 天一次）
```

#### 冬季（12-2 月）：保溫期
```
目標溫度：白天 23-25°C，夜間 15-17°C
濕度目標：60-70%
保溫策略：
  - 使用透光保溫幕（晚間覆蓋）
  - 側窗白天只開上 1/3，減少熱流失
  - 避免露水凝結（微開頂窗）

產量注意：
  - 冬季日照短，產量比夏季低 15-20%
  - 選用低溫適應品種（如陸奧）
```

## 第 3 章：定植與密度管理

### 定植準備

#### 土壤改良（定植前 2-3 週）
```
基底厚度：20-25 cm 營養土

土壤配方（參考農友實踐）：
  - 泥炭土：50%（保水性、透氣性）
  - 椰纖：20%（保水性、可生物降解）
  - 珍珠石或蛭石：20%（排水、透氣）
  - 有機質（堆肥或發酵雞糞）：10%

營養元素補充：
  - 複合肥 NPK（10:10:10）：500 kg/1000 m²
  - 鎂肥（或複合鎂肥）：100 kg/1000 m²
  
pH 調整：6.0-6.5 為理想範圍
  - 如 pH > 7.0，加入硫磺粉降低
  - 如 pH < 5.5，加入石灰粉提高
```

### 定植時機與密度

#### 嘉義地區的定植時間表
| 季節 | 育苗完成 | 定植時間 | 採收期 | 下棚時間 |
|------|---------|---------|--------|---------|
| 春季 | 2月中 | 3月中旬 | 5月中-8月中 | 8月底 |
| 夏季 | 5月中 | 6月中旬 | 8月中-11月中 | 11月底 |
| 秋季 | 8月中 | 9月中旬 | 11月中-2月中 | 2月底 |

#### 標準密度配置（以 1,800 m² 為例）
```
定植方式：畦式種植

行距：90-100 cm（便於人工操作）
株距：30-35 cm（0.25-0.3 m² 一株）
每 1,000 m²：3,300-3,700 株
每 1,800 m²：5,900-6,600 株

建議採用此密度的原因：
  ✓ 避免過密導致通風不良、病害易發
  ✓ 避免過稀導致單位面積產量過低
  ✓ 便於人工採收、病害巡檢
```

#### 定植操作步驟
```
第 1 天：準備
  - 溫室澆透水一次
  - 確認苗齡 6-7 週，展開葉 4-5 片
  - 準備定植工具（移植鏟、澆水瓢）

第 2-3 天：定植
  1. 每株挖坑深度 15-20 cm
  2. 將育苗盤中的秧苗連同根團取出
  3. 植入坑中，覆土至原有深度（不要蓋過葉片）
  4. 輕輕壓實，澆透定根水

定植後管理：
  - 前 3-5 天保持土壤濕潤（50-60%）
  - 白天遮光 30%（避免新植株受光灼傷）
  - 活著率預期：> 95%
```

## 第 4 章：肥水管理

### 水肥管理的核心原則

小黃瓜是「高耗水」作物，但對肥料敏感。實地農友的成功經驗表明：**精準的滴灌 + 分階段施肥** 是提高產量和品質的關鍵。

### 灌溉制度

#### 灌溉設備選擇
```
推薦方案：滴灌帶（單根滴頭間距 30-40 cm）

優點：
  ✓ 節水 30-40%（相比大水漫灌）
  ✓ 可整合肥料施用（肥水一體化）
  ✓ 減少莖葉觸水（降低病害）
  ✓ 自動化易實現

成本：
  - 滴灌帶購置：約 1.5-2 萬元
  - 主水管、支管：約 1 萬元
  - 水表、過濾器：約 0.5 萬元
  總計：約 3-3.5 萬元（一次性投資）
```

#### 月份灌溉計劃

| 月份 | 氣候 | 灌溉頻率 | 每次用水量 | 方式 |
|------|------|---------|-----------|------|
| 3-4月 | 氣溫回升，蒸發增加 | 1-2 天/次 | 15-20 mm | 滴灌 |
| 5月 | 高溫乾燥 | 1 天/次 | 20-25 mm | 滴灌 + 局部噴灑（降溫） |
| 6-8月 | 高溫多雨 | 降雨後延遲灌溉 3-5 天 | 10-15 mm | 滴灌（需排水） |
| 9-10月 | 溫度穩定 | 1-2 天/次 | 15-20 mm | 滴灌 |
| 11-2月 | 低溫乾燥 | 2-3 天/次 | 10-15 mm | 滴灌（晨間為佳） |

**澆灌時機**：清晨 6-8 點為最佳（此時植株吸收快，病害風險低）

### 分階段肥料管理

小黃瓜全生育期約 12-14 週，分為 4 個肥料施用階段：

#### 第 1 階段：定植-開花前（1-3 週）
**目標**：促進根系發育和葉片生長

```
肥料類型：高磷鉀肥（NPK = 5:10:10）
施肥方法：
  - 定植時基肥已施（見 3.1 章）
  - 定植後第 2 週開始淋施液肥
  - 配合灌溉時施用（肥水同步）
  
施肥量參考：
  - 週施肥量：0.5-1 kg 複合肥 / 1,000 m²
  - 分 2-3 次施用（每次 0.2-0.4 kg）
  
營養液配方（自製方案）：
  - 複合肥（NPK 5:10:10）：1 kg
  - 清水：1,000 L
  - EC 值：0.6-0.8 mS/cm
```

#### 第 2 階段：開花-初期結果（4-7 週）
**目標**：支持開花結果，促進初期果實肥大

```
肥料調整：NPK = 8:8:12（增加鉀肥比例）
施肥量：
  - 週施肥量：1-1.5 kg 複合肥 / 1,000 m²
  - 分 3 次施用（每次 0.3-0.5 kg）

微量元素補充：
  - 鎂肥（Mg）：開始週施 100-150 g / 1,000 m²
  - 鈣肥（Ca）：預防裂果，週施 100-150 g / 1,000 m²
  
營養液 EC 值：0.8-1.0 mS/cm

施肥頻率：每 3-4 天一次（結合灌溉）
```

#### 第 3 階段：盛果期（8-11 週）
**目標**：維持高產量，延長盛果期

```
肥料調整：NPK = 6:6:14（高鉀，降低氮肥避免貪青）
施肥量（最高）：
  - 週施肥量：2-2.5 kg 複合肥 / 1,000 m²
  - 分 3-4 次施用（每次 0.5-0.7 kg）

微量元素加強：
  - 鎂肥：週施 150-200 g / 1,000 m²
  - 鈣肥：週施 150-200 g / 1,000 m²
  - 硼肥（B）：預防果實苦味，月施 50 g / 1,000 m²

營養液 EC 值：1.0-1.2 mS/cm（高點）
施肥頻率：每 2-3 天一次
```

#### 第 4 階段：後期衰退期（12-14 週）
**目標**：延遲衰退，提高可採收期

```
肥料調整：NPK = 4:4:16（極高鉀，促進養分向果實轉移）
施肥量（逐週遞減）：
  - 週施肥量：1.5-2 kg 複合肥 / 1,000 m²
  - 分 2-3 次施用

微量元素：
  - 鎂肥：週施 100-150 g / 1,000 m²
  - 鈣肥：週施 100-150 g / 1,000 m²

營養液 EC 值：0.9-1.1 mS/cm
施肥頻率：每 3-4 天一次

採收策略：
  - 第 14 週末開始減肥，停止灌溉
  - 目的：加速下棚，準備下一季輪作
```

### 水肥管理的常見問題

| 問題 | 症狀 | 原因 | 解決 |
|------|------|------|------|
| 缺鎂 | 葉脈綠，葉肉褪綠 | 施肥過量導致拮抗 | 補充鎂肥 150-200 g/1000m² |
| 缺鈣 | 果實頂端軟爛（臍腐病） | 灌溉不穩定，鈣吸收差 | 穩定灌溉，加鈣肥 |
| 過度施肥 | 植株貪青，挑花落果 | N 肥過多 | 降低 N 肥，增加 K 肥 |
| 灌溉不足 | 果實短，品質差 | 缺水導致吸收不足 | 按表 4.1 執行灌溉 |

## 第 5 章：病蟲害防治

### 嘉義小黃瓜的主要病蟲害

小黃瓜在設施環境下容易遭受高濕病害和特定蟲害侵襲。實地農友的防治經驗表明：**預防 > 防治**。

#### 病害防治圖表

**高發期病害（春夏季）**

| 病害名稱 | 高發季 | 症狀 | 防治方法 |
|---------|--------|------|---------|
| 白粉病 | 春季（3-5月） | 葉表白粉，嚴重時全葉覆蓋 | 預防：通風 + 降濕；發病：2-3天噴硫磺粉或防治藥劑 |
| 疫病 | 雨季（6-8月） | 莖部出現水漬狀病斑，快速蔓延 | 預防：控濕 + 排水；發病：立即移除病株，噴布銅劑 |
| 炭疽病 | 高溫多濕（7-9月） | 果實出現圓形褐斑，凹陷 | 預防：清葉（移除老葉，透風）；發病：噴甲基托布津 |

**低發期病害（秋冬季）**

| 病害名稱 | 高發季 | 症狀 | 防治方法 |
|---------|--------|------|---------|
| 病毒病 | 全年（春秋尤甚） | 葉片皺縮、畸形，全株衰弱 | 預防：滅蚜、滅葉蟎；發病：無藥可救，需移除病株 |
| 露菌病 | 冬季（12-2月） | 葉片黃化，背面白色粉狀物 | 預防：通風、控濕；發病：噴錳鋅及戰士 |

#### 蟲害防治圖表

**主要害蟲**

| 害蟲名稱 | 高發季 | 危害 | 防治方法 |
|---------|--------|------|---------|
| 瓜實蠅 | 結果期全年 | 蛀食果實，導致腐爛損失 | 套袋（黃紙套）+ 誘蠅罐（醋 + 糖 + 藥劑） |
| 蚜蟲 | 春秋季（3-5月、9-11月） | 吸收植株汁液，傳播病毒 | 黃粘板誘殺 + 週噴肥皂水或礦物油 |
| 葉蟎 | 高溫乾燥期（5-8月） | 葉片退綠，嚴重時脫落 | 提高濕度 + 噴施克蟎寧（交替使用） |

### 月份防治計劃表

```
【春季 3-5 月】— 預防白粉病、疫病、蚜蟲
  每週巡檢：檢查葉片下表面、莖部、新芽
  定期防治：
    - 白粉病預防：噴硫磺粉（每 7-10 天一次）
    - 蚜蟲防治：黃粘板 + 肥皂水噴霧（每 10-14 天）
  環控配合：保持通風、濕度 < 70%

【夏季 6-8 月】— 預防炭疽病、疫病、葉蟎
  每週巡檢：特別檢查果實、老葉
  定期防治：
    - 炭疽病預防：移除老葉（保持通風），噴甲基托布津
    - 疫病防治：排水良好，不淋頂部，噴銅劑（10-14 天）
    - 葉蟎防治：提高濕度至 70-75%（噴水），必要時噴克蟎寧
  環控配合：遮光 50%、充分通風、晚間微噴降溫

【秋季 9-11 月】— 預防病毒病、蚜蟲、瓜實蠅
  每週巡檢：檢查蚜蟲密度、蠅蟲
  定期防治：
    - 蚜蟲防治：關鍵害蟲，黃粘板密集放置（每 100 m² 置 10-15 個）
    - 瓜實蠅防治：結果期開始套袋，誘蠅罐每 100 m² 置 3-5 個
    - 病毒病預防：滅蚜是關鍵
  環控配合：保持通風、濕度 65-75%

【冬季 12-2 月】— 預防露菌病、病毒病
  每週巡檢：檢查葉片黃化程度
  定期防治：
    - 露菌病防治：保持通風（避免露水凝結），噴錳鋅（7-10 天）
    - 病毒病預防：持續滅蚜
  環控配合：側窗白天開啟，夜間微開頂窗（防露水）
```

### 化學藥劑的安全使用

#### 推薦藥劑清單（低毒或生物農藥優先）

| 病蟲害 | 推薦藥劑 | 稀釋倍數 | 施用間隔 | 安全預期期 |
|--------|---------|---------|---------|-----------|
| 白粉病 | 硫磺粉 | 原粉 | 7-10 天 | 0 天（食用可安全） |
| 蚜蟲 | 肥皂水 | 0.3% | 7 天 | 1 天 |
| 蚜蟲 | 克蟎寧 | 1,500 倍 | 10-14 天 | 3-5 天 |
| 炭疽病 | 甲基托布津 | 1,000 倍 | 10-14 天 | 7 天 |
| 疫病 | 銅劑 | 1,000 倍 | 7-10 天 | 3-5 天 |

**使用原則**：
- 避免連續使用同一種藥劑（容易抗藥性）
- 採輪替策略：如硫磺 → 肥皂水 → 克蟎寧 → 硫磺
- 施用時間：傍晚 5 點後施用（避免日灼）
- 採收前 7 天停止化學藥劑施用

#### 通路 3：直銷宅配（自營品牌）

**特點**：

## 第 9 章：常見問題與排除

### Q1：產量比預期低，如何診斷？

**症狀**：產量只有 50-60 kg/1000m²，預期 80+ kg/1000m²

**可能原因**及解決方案：

```
原因 1：灌溉不足
  診斷：葉片午間萎蔫，土壤乾燥 < 40%
  解決：按照第 4.1 章月份灌溉計劃，確保 EC 值 0.8-1.2

原因 2：肥料不足或失衡
  診斷：老葉黃化，新葉淡綠
  解決：檢查肥料施用量是否達表 4.2，補施高鉀肥

原因 3：光照不足
  診斷：莖長細，節間長，果小
  解決：檢查遮光幕使用時間，夏季以外避免長時間遮光

原因 4：病害爆發
  診斷：葉片出現斑點、扭曲，果實不正常
  解決：立即執行第 5 章病蟲害防治，隔離病株

原因 5：溫度不適
  診斷：花多不結果，果實畸形
  解決：檢查白天溫度是否保持 25-28°C，避免 > 30°C
```

### Q2：果實表皮有黑點或褐斑，如何處理？

**症狀**：收穫期果實出現圓形褐斑，影響商品性

**診斷與處理**：

```
症狀 1：圓形褐斑，邊界明顯
  → 炭疽病
  處理：立即移除病果和病葉，啟動殺菌劑防治

症狀 2：水漬狀斑點，快速擴大
  → 疫病
  處理：檢查排水系統，減少灌溉頻率，噴銅劑

症狀 3：表皮粗糙，顏色淡斑
  → 環境因素（日灼、藥害）
  處理：調整遮光、施藥時間（避免高溫時施用）
```

### Q3：採收後黃瓜快速軟化，如何延長保存期？

**症狀**：採收 2-3 天後，黃瓜開始軟化、失水

**解決方案**：

```
採後處理優化：
  1. 預冷至 10-12°C（冷藏保存 > 5 天）
  2. 避免與成熟水果混放（乙烯催熟）
  3. 運輸途中保持 90%+ 濕度（防失水）
  4. 批發市場採購後立即販售（不囤貨）

包裝改善：
  - 使用透氣紙盒代替密閉容器
  - 內墊通風設計（防水分凝結腐爛）
  - 避免擠壓（傷表皮加速軟化）
```

## 第 10 章：檢查清單與下棚準備

### 完整的季前檢查清單

在每季開始前（3-5 日），執行以下檢查：

```
【設施檢查】
□ 溫室骨架是否有生銹、破損？
□ 塑膠布是否有破洞、老化？
□ 通風窗是否正常開閉？
□ 覆蓋幕軌道是否順暢？

【灌溉系統檢查】
□ 主水管、支管是否有漏水？
□ 滴灌帶是否堵塞？（抽水測試）
□ 肥料罐、過濾器是否乾淨？
□ 水表讀數是否正常？

【土壤檢查】
□ pH 值在 6.0-6.5 之間？（取樣檢測）
□ 土壤孔隙度是否良好？（按壓測試）
□ 是否有鹽分積累跡象？

【環控設備檢查】
□ 溫濕度計是否正常運作？
□ 通風機、風扇是否運轉順暢？
□ 自動控制系統（如有）是否校準？

【防蟲防病準備】
□ 黃粘板是否備足（15-20 個 / 100 m²）？
□ 農藥庫存是否充足？
□ 安全防護用具（手套、口罩、眼鏡）是否備妥？
```

### 下棚流程（採收結束後）

```
第 1-2 週：植株移除與清理
  1. 採收最後一批果實
  2. 移除全部植株（拔根、清運）
  3. 清除殘留肥料、肥料罐
  4. 掃除落葉、雜物

第 3 週：土壤消毒與改良
  1. 土壤翻犁（深度 20 cm），混入秸稈
  2. 曝曬 10-15 天（天然紫外線消毒）
  3. 添加新有機質（堆肥、蕈渣 10 公噸 / 1000 m²）
  4. 調整 pH（如需要）

第 4 週：設施檢修
  1. 檢查維修塑膠布破洞
  2. 清潔通風窗、覆蓋幕軌道
  3. 檢修灌溉系統
  4. 更新老舊黃粘板、誘蠅罐

第 5 週：準備下季
  1. 訂購種苗（至少 2 週前）
  2. 備妥肥料、農藥
  3. 進行土壤施肥（複合肥 500 kg / 1000 m²）
  4. 準備育苗資源
```
