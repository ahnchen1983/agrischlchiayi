## 指南簡介

農作物的病蟲害防治，預防勝於治療。本指南教農友如何以「植物放大鏡」的觀察視角，準確識別常見的病徵、蟲害與生理障礙，做到早期發現、早期防控，減少農場損失。

## 病徵觀察的基礎

### 觀察視角

- 整體農場環境（氣候、肥培、灌溉）與病徵的關聯
- 植物個體症狀（葉片、莖稈、根部、果實）的系統檢查
- 病原體（病菌、病毒、蟲害、營養缺乏）的識別邏輯

### 診斷工具與方法

- 放大鏡、相機等簡易診斷工具
- 樣本採集與記錄的標準方法
- 何時尋求專家協助（田間診療服務、農業改良場）

## 常見作物病徵識別

### 葉部症狀識別

**病菌性葉斑**
- 特徵：葉片上出現有邊界的褐色/黃色斑點，中央可見同心圓紋
- 常見作物：番茄、黃瓜、草莓
- 防控：移除病葉，改善通風，避免葉面過濕

**病毒感染**
- 特徵：葉片出現斑駁、皺縮、花葉紋，植株矮化
- 常見作物：瓜類、豆類、馬鈴薯
- 防控：防止蚜蟲傳播，拔除病株隔離

**營養缺乏**
- 特徵：
  - 缺氮：葉片淡綠，下位葉先黃化
  - 缺磷：葉片紫紅，植株生長遲緩
  - 缺鉀：葉片邊緣焦枯，先從下位葉開始
- 防控：根據症狀進行葉面或土壤追肥

### 莖部與根部症狀

**立枯病/根腐病**
- 特徵：苗期莖部收縮、變黑，根部腐爛，植株枯死
- 常見作物：蔬菜、瓜類
- 防控：改善排水，使用無菌介質，輪作

**蟲害蛀蝕**
- 特徵：莖部有蛀孔，莖內有蟲道，植株萎蔫
- 常見蟲害：斜紋夜蛾、番茄夜蛾、鑽蛀性害蟲
- 防控：物理防除（套袋、黏板），生物防治

### 果實症狀

**灰黴病/白粉病**
- 特徵：果實表面覆蓋灰白色霉層
- 常見作物：草莓、番茄、甜椒
- 防控：改善通風，移除病果，嚴控濕度

**日燒症**
- 特徵：果實向日照面出現水漬狀或褐色斑塊
- 原因：高溫、日照強、水分不足
- 防控：適當遮陰，增加灌溉，補充鈣肥

## 診斷決策流程

```
發現異常症狀
    ↓
【步驟1】觀察症狀位置和分佈
    ├─ 全株均勻分佈 → 可能是環境或營養問題
    ├─ 局部或下位葉 → 可能是病蟲害
    
【步驟2】檢視病徵特徵
    ├─ 是否有病原體跡象（黴層、蟲體、斑點邊界）
    ├─ 症狀發展速度（快速蔓延 vs 緩慢進行）
    
【步驟3】對比常見病徵資料庫
    ├─ 該作物常見病害有哪些
    ├─ 季節與環境條件是否吻合
    
【步驟4】決定行動
    ├─ 如果高度確定 → 立即防控
    ├─ 如果不確定 → 採樣送檢或尋求診療服務
    ├─ 如果是環境問題 → 調整灌溉、施肥、通風
```

## 防治策略層級

### 【預防】非藥劑防治（第一優先）

- 選用抗病品種
- 改善栽培環境（通風、日照、排水）
- 輪作與清園（移除病殘體）
- 使用無病種苗或穗條
- 定期檢查與早期拔除病株

### 【防控】一旦發現病蟲害

**物理防治**
- 人工移除病葉、病果
- 黏板、套袋、粘蟲防護網

**生物防治**
- 利用天敵（瓢蟲、寄生蜂）
- 微生物製劑（蘇力菌、木黴菌）

**化學防治**（必要時）
- 優先選用低毒性藥劑
- 輪流使用不同機轉的藥劑，避免產生抗性
- 遵守安全採收期
- 記錄用藥時間與藥劑名稱

## 記錄與學習

### 田間日誌的重要性

農友應建立簡易的田間病蟲害記錄，包括：
- 日期、天氣條件（溫度、濕度、降雨）
- 症狀首次發現時間與位置
- 採取的防控措施與效果
- 用藥類型與日期（如果有）

**好處**：
- 發現病害規律，預測下次發生時間
- 評估防控措施的有效性
- 為下年度栽培規劃提供依據
- 累積個農場的防控經驗數據庫

## 尋求專家協助

### 何時聯絡農業改良場

- 症狀無法確定診斷
- 常見防控方法無效
- 出現未曾見過的病害
- 需要進一步的實驗室檢驗

### 提供給診療人員的信息

採樣送檢時，應提供：
1. 清楚的症狀照片（遠景、近景）
2. 病害發生時間、發展速度
3. 植株部位、田間發生率（% 或株數）
4. 最近的灌溉、用肥、用藥記錄
5. 其他症狀的植株樣本（若有）

詳細的資訊能幫助診療人員快速準確診斷。
