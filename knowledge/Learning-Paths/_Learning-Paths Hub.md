## 學習路徑推薦

本頁面依據不同角色與經驗程度，推薦系統性的學習路徑。農友可依自身需求選擇適合的路線，循序漸進建立完整的農業知識體系。

## 路徑一：新手農民入門

適合剛踏入農業領域的新手，從基礎知識開始建立穩固的農業根基。

**建議學習順序：**

1. [[土壤基礎知識]] - 認識農業的根本
2. [[水資源管理入門]] - 掌握灌溉基礎
3. [[田間管理技巧]] - 學習田間實務操作
4. [[有機農業實務]] - 了解友善農法
5. [[農業補助申請指南]] - 善用政府資源起步
6. [[農場財務規劃]] - 建立財務管理觀念

## 路徑二：經驗農民進階

適合已有栽培經驗、希望提升技術與導入新科技的農友。

**建議學習順序：**

1. [[小番茄栽培技術]] 或 [[百香果種植指南]] - 精進特定作物技術
2. [[溫室設施規劃]] - 評估設施化升級
3. [[農業物聯網入門]] - 導入智慧監測工具
4. [[無人機農業應用]] - 了解精準農業科技
5. [[農產品品牌建立]] - 提升產品附加價值
6. [[模範農場參訪紀錄]] - 借鏡成功經驗

## 路徑三：農場經營管理者

適合農場主或管理階層，著重經營管理與策略規劃。

**建議學習順序：**

1. [[農場財務規劃]] - 掌握經營成本與財務健全
2. [[農業人力管理]] - 建立高效團隊
3. [[農產品品牌建立]] - 建立品牌競爭力
4. [[農業補助申請指南]] - 爭取政府資源
5. [[溫室設施規劃]] - 評估設施投資效益
6. [[模範農場參訪紀錄]] - 學習標竿農場管理模式

## 路徑四：畜牧業入門

適合有意投入畜牧業或小型家禽養殖的農友。

**建議學習順序：**

1. [[家禽飼養管理]] - 家禽飼養基礎
2. [[水資源管理入門]] - 畜牧用水管理
3. [[農場財務規劃]] - 畜牧場經營成本
4. [[農業補助申請指南]] - 畜牧相關補助
5. [[農業人力管理]] - 畜牧場人員管理
