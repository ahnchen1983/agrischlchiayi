# 嘉義農業學堂

> 將嘉義縣國本學堂的農業課程，轉化為公開、開源、持續成長的農業知識平台。

## 專案目的

讓新農、轉行者、學生、農企業主能系統化學習農業知識。透過 **Graph View（知識圖譜）** 呈現課程之間的關聯，讓學習路徑清晰可視。

## 十大知識類別

| #   | 類別         | 說明                 |
| --- | ------------ | -------------------- |
| 1   | 農業基礎入門 | 新農入門的第一步     |
| 2   | 農業進階實務 | 進階栽培與田間管理   |
| 3   | 農場經營管理 | 農企業經營與領導     |
| 4   | 作物生產技術 | 栽培技術與病蟲害管理 |
| 5   | 設施農業     | 溫室與設施技術       |
| 6   | 智慧農業     | 科技應用與數位轉型   |
| 7   | 農產品銷售   | 品牌行銷與通路       |
| 8   | 計畫撰寫補助 | 政府補助與永續經營   |
| 9   | 場域交流觀摩 | 實地參訪與成果發表   |
| 10  | 畜牧健康     | 畜牧管理與生物安全   |

## 技術架構
