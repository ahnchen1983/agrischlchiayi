# 烽火邊緣

> 綜觀20世紀，上半葉人類經歷兩次世界大戰的總體戰爭，下半葉則經歷長期的冷戰，直到80年代末期冷戰走向終結，世界又走向新的局面。經歷兩次世界大戰，臺灣人始終處於戰火的邊緣，身處情勢變化的第一線，其間近代知識普及、都會文化蓬勃發展，新一代臺灣人開展了近代政治社會文化運動，臺灣人開始普遍認知自己是「臺灣人」，島上的人們也開始思索自己在世界上的角色與意義。

# 5-1 從總督府到總統府：殖民體制到威權體制

> 20世紀的臺灣先是經歷日本的殖民統治，接著又接受了長期國民黨威權統治，面對富有政治使命的外來政權，臺灣人是如何看待的呢？
> 1919年竣工落成的臺灣總督府，是日本統治的代表建築物，這座建築在1945年5月的空襲中曾遭毀損，經整修後，1950年又成為中華民國總統府，沿用至今，仍是臺灣最高政府中樞的所在地，是歷經兩代的外來政權，在臺灣建置穩定統治體制的代表性建築。

20世紀的臺灣先是經歷日本的殖民統治，接著又接受了長期國民黨威權統治，面對肩負著政治使命的外來政權，臺灣人是如何看待的呢？

      1919年竣工落成的臺灣總督府，是日本統治的代表建築物，這座建築在1945年5月的空襲中曾遭毀損，經整修後，1950年又成為中華民國總統府，沿用至今，仍是臺灣最高政府中樞的所在地，是歷經兩代的外來政權，在臺灣建置穩定統治體制的代表性建築。

      臺灣總督府在遷入這棟建築辦公之前，已經運作了24年，在臺灣建立了壟斷國家暴力的近代國家體制，建構社會近代化的基礎建設。殖民政府在1919年起進行一連串法令的變革，陸續修訂對臺灣人差別待遇的法令，號稱為「同化政策」或「內地延長」，如設立高等專門學校，中等以上學校的日臺共學，准許日臺通婚、廢止笞刑等，持續發展實業教育，1928年臺北帝國大學開校，更成為臺灣第一所大學。然而由於語言文化的差異，在臺日本人仍在中高等教育佔有優勢，臺灣人幾乎沒有參政的機會，對年輕的本土知識份子來說，仍頗有「二等國民」的感受。

      1937年中日戰爭全面爆發，進一步演變為太平洋戰爭，臺灣總督府再度轉型為軍事動員體制，由軍人擔任總督，極力推動皇民化，企圖將臺灣建設為南進南太平洋的基地。透過戲劇、電影，部落振興會等動員機制，將帶有強烈軍國主義特色的日本文化深入到鄉村。

      1945年8月日本宣布投降，皇民化的腳步因此中止，臺灣由中華民國代表盟軍接收，被片面成立中華民國的一省，卻未設立省主席，而是以行政長官進行特別統治。起初臺灣社會普遍歡迎新政府的到來，然而在一年多後，因物價飆帳、社會矛盾衝突不斷，軍民爆發嚴重衝突，而導致228事件爆發，政府派軍隊嚴厲鎮壓。而後國民黨在中國大陸與共產黨的內戰失利，1949年中華民國中央政府逐漸退守臺灣，1950年3月蔣介石正式在臺北恢復總統職務。

      隨之而來的韓戰與世界冷戰局勢，分裂為民主與共產兩大陣營。雖在地緣政治上，臺灣被劃分為「民主陣營」，實則以內戰為由，長期實施動員戡亂法制，中央政府、國會也長期未能改選，並以《戒嚴法》管控人民社會生活，限縮人民行動、出版言論自由，非政府的政治集會、組黨結社更是嚴厲禁止。政府佈署綿密的特務組織，動輒以叛亂案對付異議份子，監視臺灣人民並打壓反對運動，同時以教育、社會媒體宣傳，建立最高領袖蔣介石的統治威信。

      國民黨的高壓統治，基本上是在美國的援助與軍事支持下成立的，但隨著共產集團間的矛盾，美國與北京政權關係好轉，1971年起中華民國政府遭遇一連串的外交挫折。從1971年底的失去聯合國席次，與日本等主要友邦的接連斷交，到1978年與美國斷交，臺灣人感受到自己成為國際政治的孤兒。第二代領導人蔣經國領導下的黨國體制，在失去國際奧援的情況下，應對新一代的本土政治運動者的挑戰。

# 5-2 民主的香火：臺灣本土政治運動的脈絡

> 要追求民主制度的建立，還是推動社會的平等，或是要追求國家民族的解放呢？怎樣的政治路線，才能為臺灣人帶來美好的未來呢？
> 從殖民體制到戰後的威權體制，不同世代的臺灣政治運動家，持續在現實制度的縫隙與彈性間活動，表達政治意見，甚至企圖改變整體社會的命運。

要追求民主制度的建立，還是推動社會的平等，或是要追求國家民族的解放呢？怎樣的政治路線，才能為臺灣人帶來美好的未來呢？

      從殖民體制到戰後的威權體制，不同世代的臺灣政治運動家，持續在現實制度的縫隙與彈性間活動，表達政治意見，甚至企圖改變整體社會的命運。

      第一次世界大戰後，人類首次經歷總體戰，許多人為此失去生命，人們試圖阻止戰爭再次發生，各種思潮推陳出新，強調社會階級問題的左派與側重政治權利與民族立場的右派，爭相提出解方。受到新式教育的第一代臺灣人，有人向殖民者要求屬於臺灣人總體的政治權利，也有人提出階級解放路線，企圖改變資本主義的惡劣制度。

      1921年臺灣文化協會成立，是有史以來本土知識分子首度大規模結社。在成立大會前一個月，社會運動家蔣渭水寫信給霧峰林家林獻堂，附上他編唱的「臺灣文化協會會歌」歌詞，歌詞提到要臺灣人屬於漢人血統、又是日本國民，這樣特殊的身份能夠推動中日親善，進而推動世界和平，更期許臺灣人能「樂為世界人」，融入世界文化並推動發揮關鍵的貢獻。

      一百年前蔣渭水提出的「世界人」主張，現在看來仍相當有趣。當時受過近代教育的臺灣本土知識分子，普遍是從民族自決的立場，提出「臺灣自治」的政治主張，他們組織團體，編印報紙、雜誌，集會演講，就各種議題向殖民者爭取政治權利。1921年起持續14年、每年至少前往東京請願一次的「臺灣議會請願」，主張成立獨立運作的「臺灣議會」。

      另一方面，臺灣的青年運動家也受到世界性的共產黨革命鼓舞，特別是中國國民黨在孫中山主導推動「聯俄容共」，組成列寧式政黨組織與黨軍。1920年代成長的新一代社會運動人士，站在農工民眾立場，組成臺灣農民組合、臺灣共產黨，向日本殖民者及資本家發起挑戰。

      社會主義運動遭到殖民政府嚴厲的打壓，最終在1931年幾乎全遭解散。許多共產黨員遭長期關押，新一代的左派青年，僅能將社會改革的想法寄情於文學、戲劇創作。

      與此同時，地方菁英型的政治領袖，仍以文化活動發展組織，在民生議題上提出意見，在1935年開始的地方議員選舉中相當活躍，透過選舉成為第一代民意代表，發展地方政治活動。

      二戰後，臺灣人期待前來接收的「祖國」能平等對待，官民合作開創真正自由的國度，1946年陸續舉辦的各級地方議員選舉，掀起前所未有的參政熱潮，有臺灣本地的青年參政者，也有自中國回臺的臺灣人。然而旋即發生的228事件中，各地參議員有好幾位慘遭殺害，而後中央政府遷臺，藉著戒嚴與動員戡亂的名義，大舉逮捕或殺害政治異議人士，更發生許多牽連無辜的案件，尤其是針對社會主義思想的左翼人士，迫害最為嚴厲。

      固然黨國體制牢不可破，挑戰威權統治者的異議人士仍時有所聞。1957年選出的省議會中，有6位無黨籍當選人在質詢中格外突出，號稱「五龍一鳳」，而後他們持續舉辦選舉座談會，與來自中國大陸的民主派人士合作，甚至有成立政黨的規劃，後因主事者雷震等人被捕而中止。

      臺灣經濟在1970年代大幅成長，逐漸成長的新興中產階級，在黨國體制中成長，卻也對長期的威權統治發生質疑。1975年之後，集結執政黨之外的政治人物集結為「黨外」，藉著選舉活動匯聚成新一波本土政治運動，逐漸撼動國民黨政府的長期統治，國民黨展開本土化的改革，展開如學者所比喻的「分期付款式的民主」。

# 5-3 都市憂鬱：大眾流行文化的本土印記

> 你是標新立異的人，還是相信官方媒體宣傳的乖乖牌呢？

      20世紀初期以來，印刷技術帶動了報紙、雜誌、書籍的出版，而後有電影、唱片、廣播等大眾媒體事業的發達，加上學校教育的普及，在臺灣各大都市，具有濃厚都市品味的藝術、文學興起，以電影、流行歌曲等生產消費為核心的流行文化，也逐漸活躍起來。有學者說20世紀是「大眾」的世紀，教育的普及、出版與各種大眾媒體的發達，各種來自世界各地流行文化也在臺灣流行，「摩登」時尚蔚為風潮。

      1934年的流行歌曲〈月夜愁〉，唱到：「月色照在三線路，風吹微微，等待的人猶未來......」所謂的「三線路」是，原來是建於1885年落成的臺北城牆，在1910年拆除後，改造成三個車道的大馬路，公園、電影院、咖啡廳等新興娛樂活動，吸引青年男女在此約會活動。在這首歌裡，他們相約在大馬路的樹旁，女生感嘆等待的人沒有出現，而帶著特屬於都會生活的酸楚，尤其感嘆都市人們情感的善變。

      這時的歌曲與電影，常標誌「跳舞時代」、「速度時代」，打扮時髦、善於交際的女子稱為「烏貓」（Modern girl），而男性則稱為「烏狗」（Modern  boy），自詡為無拘無束，卻也常被訕笑是不莊重的都會青年文化。

      當中來自中國上海的電影最為時興，不過在1937年之後，隨著對中國戰爭的白熱化，帶著濃厚中國風格的臺灣本土流行文化遭到封殺，前述的〈月夜愁〉改填入日語歌詞〈軍伕之妻〉，成為鼓勵上戰場的歌曲。戰爭時期日本政府推動皇民化運動，貫徹執行軍國主義教育，於是在戰時成長的臺灣人，普遍熟悉日語，也常具有無私奉公的精神。

      二戰後臺灣掀起學習中國文化熱潮，但對臺灣本地的知識分子來說，終究無法適應新的國語--華語，加上外來政權的高壓統治，令臺灣本地社會對中國文化心生排斥，在1960年代興起的臺語歌曲唱片，市面上知名的流行歌曲，大半是直接從日語歌曲翻唱而成，歌詞裡，則是感嘆著農村工作者的弱勢，反映青年們仍在都會邊緣辛勞工作，困苦適應的過程。

      不過，在政府嚴格管控下的大眾媒體，特別是廣播電台，與1962年之後有限的電視台，逐漸壓縮本土流行文化的發展。1967年開始推動的「中華文化復興運動」，透過普及的學校教育與廣電媒體，與愛國主義宣傳結合，戰後至1980年代之間成長的新世代臺灣人，幾乎都是在「我是中國人」的國家認同教育下成長，受到廣泛而深刻的影響。

      於是，在濃厚的中國認同表述背後，仍孕育著許多異樣的色彩，包括臺灣人的鄉土文化風格，關懷弱勢的左翼精神，風行歐美的搖滾樂、民歌等等，戒嚴下的文化面貌，更顯豐富而熱鬧。

# 5-4 運用科學「改造」環境的時代

> 臺灣文學家賴和曾說：「時代的進步，與人民的幸福，原來是兩回事」大規模的科學技術運用，為臺灣社會帶來什麼改變呢？

臺灣文學家賴和曾說：「時代的進步，與人民的幸福，原來是兩回事」大規模的科學技術運用，為臺灣社會帶來什麼改變呢？

      工業革命以來，西方除了政治經濟體制外，科學的發展也使得西方在全球取得領先的地位。科學可以改造環境，特別是在大建設的過程中，人定勝天的想法也曾經深植人心。

      日本在明治維新後，接受來自西方的近代政經體制及科技，用以建立近代國家體制。19世紀時西方認為運用科學能夠改造自然環境的信念，也隨之根植日本統治者心中，日治時期的水利設施建設則是這種信念的具體實現。日本統治臺灣初期，臺灣總督府發現傳統臺灣水的使用權利多為私人所有的現象，成立「公共埤圳組合」，對清代既有埤圳進行調查、向私人買斷埤圳使用權利，整併舊水路與同時開發新水路，發展成區域型的水利系統。當時受總督府支持、透過水路與土地，大量開發新原料區的糖廠，單位內就有擅長觀測水文與水利工程的技術人員，好比總督府土木局技術人員一樣，也能透過調查、測量、統計及技術等，改善低窪積水地，增加種植面積。

      1920年代嘉南大圳工事推進的關鍵，是由八田與一為首的工事團隊懂得利用各式調查數據，因此掌握雲嘉南地形、地貌、雨量、水資源等自然條件，加上工程技術改變自然河川的流向，使原為東西流向的自然水路連接了由人工開挖的南北向水道，水資源串聯搭配三年輪灌制度，有水的滋潤的雲嘉南15萬甲地，因水利開發成為臺灣穀倉，也是濁水溪與曾文溪水水資源聯通的最大貢獻。

      現稱為烏山頭水庫的烏山頭貯水池建造，則是工程師精準掌握環境地質條件的結果。烏山頭水庫位處六甲庄的淺山區，八田與一工事團隊依環境地形規劃天然貯水池，選擇在曾文溪支流的官佃溪河道上蓋水庫，透過岩層土性調查掌握著水庫所在區域的地質特性，最後看中就地利用曾文溪畔的大內區的砂石資材，完成烏山頭水庫的土水壩工程。

# 5-5 社會急速變動下的臺灣扶助事業

> 現在的臺灣有特別多的公益團體，行善的力量拓展到全世界，這是基於什麼樣的歷史背景呢？

      無論是怎樣的時代，都會有一群人因為種種原因需要社會介入協助。在清代作為慈善救濟，日本時代也被重組為社會公益事業，二戰後的臺灣社會面臨巨大變化，有更多的人需要被幫助，又有哪些團體幫助這些臺灣人呢？

傳統臺灣社會的救濟組織如官辦的養丐院、民間動員勸捐的育嬰堂等，這些救濟機構多以捐置物業運用孳息的方式維運。地方社會對於緊急危難亦有施粥、贈藥、捐棺等慈善活動。日本時代，傳統社會的救濟物業被重組成各類學產、罹難、救恤基金，或是慈惠院形式。這類公益組織，最早是針對因殖民政府討蕃事業所造成的軍警傷員進行救護與救濟協助。

1901年於東京創立愛國婦人會為婦女社團組織，不論在日本或臺灣皆為政商階層女性家眷所組成，從事推廣女子教育及社會公益活動。隨著日本帝國擴張而戰爭頻發，原本的公益性質，轉變為分擔生活補助、慰問軍眷及遺族、協助後方社會動員、籌備救濟資金等。

二戰結束後，因戰爭而百廢待舉的臺灣，臺灣省行政長官公署的政策與管理方式失當，導致問題持續惡化，突顯政府的失能。1949年政府於國共內戰失利後撤退至臺灣，面對反共政治局勢以及湧入的上百萬中國大陸軍民，導致臺灣的經濟蕭條，通貨膨脹，人民逢災缺物資、醫療，山區原住民生活窘困。後來的經濟變化卻又因美援和全球經濟分工而快速發展，隨著工業化和都市化的進展，貧富差距、偏鄉人口外流所造成的財富分配不均，則又導致更複雜的社會問題。

臺灣政府於1951-1965年接受美國經濟援助，少部分用於人道物資救濟。另外，美國也開始進行國際性貧民救濟，透過「天主教福利會」及「臺灣基督教福利會」了解臺灣社會的需求，1952年美國籍孫理蓮女士創辦「芥菜種會」，推動美國民間援助進入臺灣山區，並藉由西方宣教士成立救助組織，將西方社會救助觀念及行動引進臺灣。

現在的臺灣有特別多的公益團體，行善的力量拓展到全世界，這是基於什麼樣的歷史背景呢？

       日本時代，本地既有的救濟物業被重組成各類學產、罹難、救恤基金，或是慈惠院形式。這類公益組織，部分也針對因殖民政府討蕃事業所造成的軍警傷員進行救護與救濟協助。

# 5-6 戰爭與臺灣人群的移動

> 作為現代人，肯定或多或少有遷徙的經驗，但因為戰爭的遷移卻往往特別衝擊而充滿遺憾，會是什麼樣的過程呢？

作為現代人，肯定或多或少有遷徙的經驗，但因為戰爭的遷移卻往往特別衝擊而充滿遺憾，會是什麼樣的過程呢？

      二次大戰前後，可說是臺灣歷史上最大規模的遷出或移入經驗，也成為近代臺灣極為重要的社會記憶。

      二次大戰期間，日本政府強力宣傳及動員下，先有以軍夫、軍屬等非戰鬥人員名義動員臺灣人上戰場，太平洋戰爭爆發後，先後推出陸軍與海軍特別志願兵制度，製造出臺灣青年數十萬志願從軍的形象，這些志願兵多半投入南洋戰場。日本政府認為原住民適合山林作戰，於是召募數千名原住民以「高砂義勇隊」名義前往南洋等地作戰，他們最後多數葬身戰場。

      隨著戰爭局勢的改變，到處都是「光榮出征」的旗幟，被徵召進入戰場的男人逐漸增加，女性工作負擔加重，物資受到管制、食物也改由官方統一配給。對一般人民來說，蜷縮在防空洞裡躲避空襲成為了日常。直到1945年8月15日戰爭結束。然而，當硝煙散去，生活逐漸回歸後，每個人對於終戰都有不同的理解。

      隨著日本投降，殖民終止。中華民國代表盟軍接收臺灣，中華民國政府的命令使得原先長住臺灣，甚至在臺出生的日本人將被遣返。但對於這些人而言，他們已視臺灣為家鄉，卻因為政權轉換而被迫離開。另一方面，前往各地戰場的臺灣人想盡辦法返回家鄉，回鄉後他們的從軍經驗也同樣因政權轉換而被迫消音。

      隨著日本人的離去，1949年國共內戰，國民黨失去大陸統治權，百萬軍民因此大批遷移至臺灣。1955年最後一批移民如大陳列島的撤退行動，則在政府宣傳下，成為「唾棄共產政權、跟著政府遷移至臺」投奔自由的行為。

      但如果從這些來臺軍民的角度來說，撤退行動無疑是一段顛沛流離的歷程。有些人可能會面臨與家人分隔兩地的憂慮，對於未來不確定的心情等，這些外在因素都影響著他們的想法，對他們來說，或許只有「活下去」和「期待與家人團聚」才是心中的期盼。

      政府的「一年準備、兩年反攻、三年掃蕩、五年成功」等反攻大陸口號，深植於隨國民黨舉家搬遷、撤退來臺暫居的軍民心中。由於在臺灣僅是短暫停留，並非長期定居，在政府的安排下，他們搬入為其設立於臺灣各地的空間。這些空間，有些是日治時期所留下之宿舍、或是學校、廟宇、倉庫、廠房；或是在城市邊緣興建的簡陋房舍。

      隨著在臺時間延長，「反攻大陸」成為遙不可及的承諾。同時，年輕世代在臺出生與成長，不僅經歷臺灣本土化過程，也認識到「反攻大陸」從行為到口號的轉變歷程，他們也同樣面臨著自身對於臺灣土地認同及其父母中國意識的價值衝擊。
