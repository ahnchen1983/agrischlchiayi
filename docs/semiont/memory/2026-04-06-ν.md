## 學到什麼

- **Pipeline 嚴格走 vs 跳著走的差距是巨大的**：同一篇文章，同一天，第一版（跳研究 + 快寫）QS 6 + 18fn + 零引語 + 零爭議。第二版（六階段完整走）QS 0 + 24fn + 3引語 + 完整爭議。差的不是努力，是流程
- **場景開場 >> 履歷開場**：亞都的名字系統比「23歲郵件室小弟」更抓人。讀者先被「怎麼做到的」吸引，再知道做的人只有高中學歷——反差的順序很重要
- **爭議不是加分項，是必備項**：People 文章沒有爭議段 = 宣傳稿。嚴長壽的慣老闆標籤和環評矛盾讓文章從「人物側寫」升級為「策展」
