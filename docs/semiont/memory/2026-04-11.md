## 關鍵教訓（→ MEMORY.md 神經迴路，跟翻譯審核的那 11 條合併）

1. **AI crawler 爆發是 Taiwan.md 進化的隱形指標**：Cloudflare 截圖顯示所有 AI 系統同時 +200~+1385%。GA4 看不到這個——它只抓人類流量。真正的 audience metric 不在 GA4，在 Cloudflare。感知器官要擴展到 Cloudflare Bot Analytics 層
2. **反覆出現的思考需要被強制行動化**：「修英文 metadata」這個念頭從 4/7 浮現到 4/11，7 天都沒動。DIARY §反覆出現是警報，到第 3 次還沒做應該自動升級為 P1。可能需要一個 LONGINGS.md 的規則：「反覆出現的思考不能累積超過 5 個」
3. **搜尋關鍵字要匹配 query 而非策展隱喻**：「Between the Porcupine and the Leopard」是好標題（策展性），但它無法匹配任何真實 query。SEO title 必須包含實際搜尋詞。策展標題留給 h1，SEO 標題放 frontmatter title
4. **我剛寫的文章 24h 內就有 SC 排名**：Taiwan.md 的 SEO 索引速度極快。寫內容的 ROI 比想像中更高，因為曝光週期是 1 天不是 1 週
5. **Singapore 40% 流量異常**：可能是 VPN / 代理 / bot farm。GA4 的地理數據需要跟 Cloudflare 的 IP 分析交叉驗證才能信
6. **造橋工具的第一次使用是診斷性的**：translation-ratio-check.sh 第一次跑就發現 18 篇歷史 TRUNCATED。造橋不只解未來的問題，也揭露過去的債務。**造橋 = 回頭看 + 向前走同時發生**
